WZDx Work Zone Feed

Mobile work zones projected from John Deere Operations Center equipment, snapped to the Iowa DOT RAMS Road Network. WZDx specification v4.2.

→ feed.json  (WZDx v4.2, GeoJSON — for DOT / 511)
→ cifs.xml  (Waze CIFS, XML — for Waze for Cities)

Both regenerated every 15 minutes by GitHub Actions from the same projected data. Validate WZDx with the official WZDx validator; CIFS against your Waze for Cities onboarding.
